# Hugo Theme Azur

This is a standalone Hugo theme based on the [Azur](https://github.com/Hellf0rg0d/theme-azur) theme.
It features a dark mode, responsive design, and more.

## Installation

Inside the folder of your Hugo site run:

```bash
git submodule add https://github.com/Hellf0rg0d/theme-azur.git themes/azur
```

Then add the theme to your `hugo.toml` / `hugo.yaml` / `hugo.json`:

```toml
theme = "azur"
```

## Example Site

You can test the theme using the exampleSite included:

```bash
cd exampleSite
hugo server --themesDir ../..
```
